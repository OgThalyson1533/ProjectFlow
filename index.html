Guia de Setup — Supabase

1. Crie um projeto em supabase.com (grátis)

2. Vá em Settings → API e copie a Project URL e a anon public key

3. Clique no ⚙️ na tela de login e cole os valores

4. No Supabase, vá em SQL Editor e execute os arquivos na ordem:

sql/01_schema.sql sql/02_functions.sql sql/03_rls.sql (opcional) sql/04_seed.sql

5. Em Authentication → Providers, ative o Email provider

📄 Schema resumido — sql/01_schema.sql
